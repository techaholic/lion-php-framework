        mixed &currentItem(
)
    

    


                
        This method returns the element in the current position of the internal pointer.

It does not move the pointer in any way. If the internal pointer points beyond the  end of the elements list, currentItem returns FALSE.

                

  

    Return:  The current pointed element, or FALSE if the internal pointer points beyond the end of the elements list.
