Class: __Singleton


Class Overview

Class Overview | Descendants | Method Summary | Methods Detail
Generic Singleton base class usefull to retrieve a singleton instance from the current context.


Just inherit(extend) from __Singleton and add a getter like the following one: 

  1.  public static function &getInstance() {
    2. 

  2.    return __Singleton::getSingleton(AN_INSTANCE_ID);
    3. 

  3.  }
    4.

Note that you need to specify an instance id to retrieve the singleton instance from the current context.
The getSingleton static method will call to the getInstance method of the current __Context instance
