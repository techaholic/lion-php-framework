addParameter






    

    
    

    
 
        void addParameter(
string
$parameter_name, string
$parameter_value 
)
    

    


                
        This method adds parameters associated to the current building url.

It analyzes each one in order to know how to represent in the url building.  Note that some parameters are converted to url variables and embedded as specified  in the url regular expression. Other need to be present as it in the url.

                            

        Parameters:

                The name of the parameter
