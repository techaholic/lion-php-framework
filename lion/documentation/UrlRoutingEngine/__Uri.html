Class: __Uri


Class Overview

Class Overview | Method Summary | Methods Detail
This class represent an URI.

It is able to interpret urls, discompounding into single elements like the action code, request parameters, front controller to dispatch a request for current uri, etc...
It also make the inverse operation: build an url from a set of single elements. To make both operations, it requires a route definition (contained in a __Route instance).

i.e.:

  1. <?php
    2. 

  2.  
    3. 

  3. //Discompound an url into single elements:
    4. 

  4. $uri = __UriFactory::getInstance()->createUri('http://mydomain/login.action');
    5. 

  5. $parameters       = $uri->getParameters();
    6. 

  6. $controller_code  = $uri->getControllerCode();
    7. 

  7. $action_code      = $uri->getActionCode();
    8. 

  8. $route_id         = $uri->getRouteId();
    9. 

  9. //we can also know which front controller attends the request:
    10. 

  10. $front_controller_class = $uri->getFrontControllerClass();
    11. 

  11.  
    12. 

  12. //Compound an url from single elements:
    13. 

  13. $uri2 = new __Uri();
    14. 

  14. $uri2->setControllerCode($controller_code);
    15. 

  15. $uri2->setActionCode($action_code);
    16. 

  16. $uri2->setParameters($parameters);
    17. 

  17. $uri2->setRouteId($route_id);
    18. 

  18. $url = $uri2->getUrl(); //will return 'http://mydomain/login.action'
    19.


See:  __Route, __RouteManager, __UriFactory
