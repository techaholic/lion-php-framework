Class: __UserSession


Class Overview

Class Overview | Method Summary | Methods Detail
This class handles active roles for current user session.

Active roles are the subset of user's roles that are checked in order to grant or revoke the access to a system's resource.
Each system resource has associated a required permission to access to. The access is granted if there is any active role that contains the required permission.

See Authorization in Lion section for more information
