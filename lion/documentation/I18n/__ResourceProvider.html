Class: __ResourceProvider


Class Overview

Class Overview | Descendants | Properties Summary | Properties Detail | Method Summary | Methods Detail
This is the base class for all resource providers classes used by the LION framework.

This contains some template methods such as LoadResource/s, SaveResource/s. Only methods for connect/disconnect from the storage media, and load/store operations are specialiced in derived classes by implementing some abstract methods of __ResourceProviderBase class.
