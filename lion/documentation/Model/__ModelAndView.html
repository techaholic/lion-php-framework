Class: __ModelAndView


Class Overview

Class Overview | Properties Summary | Properties Detail | Method Summary | Methods Detail
This class contains both the model and view.

Instances of this class are usually created by actions commands. A ModelAndView class can contains the View class for rendering, or also a view name that will be used to resolve the View technology.
