__call






    

    
    

    
 
        mixed __call(

$alias, 
$parameters, array
$ 
)
    

    


                
        This overloading method permit to the ModelProxy to be called for inexistent method (i.e. GetUserName).

ModelProxy will resolve the real class and method to call, will instanciate it and reroute the call.  Of course, it will return the returned value to the caller.  This overloading method is an alias of executeModelService that makes transparent where is the real method that is called.
