    This method returns the list of events that current __EventListener is suscribed for.
  • __EventListener::receiveEvent()
    This method is called by the __EventDispatcher when a new event is raised and the current __EventListener is suscribed to it.
  • __EventListener::startToListen()
    This method starts to listen for any event raise, by calling to any callback associated to each event.
  • __EventListener::stopToListen()
    This method stop to listen for any event raise. No actions will be executed for any event.
  • __EventListener::__destruct()
    This is the destructor method. It calls to the stopToListen method
