Class: __EventListener


Class Overview

Class Overview | Descendants | Properties Summary | Properties Detail | Method Summary | Methods Detail
This is the class in charge of listen for any event raise.

This class suscribe itself to certain set of events, and for each one stores a callback. If any event is raised, then it will call to the callback method by passing a reference to the __Event.
