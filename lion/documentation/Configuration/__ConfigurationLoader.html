Class: __ConfigurationLoader


Class Overview

Class Overview | Properties Summary | Properties Detail | Method Summary | Methods Detail
This is the class in charge of loading configurations.

The context container has associated a configuration loader in order to load the configuration before start dispatching each request.

loadConfigurationFile






    

    
    

    
 
        __Configuration &loadConfigurationFile(
string
$configuration_file 
)
    

    


                
        Load an given configuration file, returning a __Configuration instance representing it.

The configuration file can be set as a relative path or absolute path.
  i.e.
 
  • config/controllers.xml (relative to the context root directory)
  • /var/www/myapp/config.xml (absolute path)
