Class: __Configuration


Class Overview

Class Overview | Inherited Properties, Constants, and Methods | Properties | Method Summary | Methods Detail
Represents a configuration file applicable to a particular application.

This class cannot be inherited.

The __Configuration class represents the merged view of the configuration settings that are applied to a logical entity, such as an application, or a Web site.
