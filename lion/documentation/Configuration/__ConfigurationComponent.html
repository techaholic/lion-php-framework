Class: __ConfigurationComponent


Class Overview

Class Overview | Descendants | Properties Summary | Properties Detail | Method Summary | Methods Detail
This is the abstract class for all configuration components.

There are 2 kinds of configuration components:

  • complex configuration components: Components that can contain more configuration components. i.e. __ConfigurationSection instances.
  • simple configuration components: Components that can not contain more configuration components. i.e. __ConfigurationProperty instances.
