Interface: __ISectionHandler


Interface Overview

Interface Overview | Method Summary | Methods Detail
This is the interface that should implement all classes in charge of resolution of configuration settings.

A __ConfigurationSection instance can have associated a class implementing the __ISectionHandler. When a section with a section handler associated is requested by the __Configuration::getSection method, the section handler will read the content of the section in order to create other components that will be returned by the getSection instead of the requested __ConfigurationSection instance. If no section handlers are associated, the original __ConfigurationSection will be returned.

This mechanism is usefull to transform some configuration pieces in terms of application components just when needed.
