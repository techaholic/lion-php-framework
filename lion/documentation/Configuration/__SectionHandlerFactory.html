Class: __SectionHandlerFactory


Class Overview

Class Overview | Method Summary | Methods Detail
This class is the factory for section handlers.

When the configuration is requested for a given section, it ask the section handler factory to get a section handler for that section. If the factory returns a section handler, the configuration will delegate on it to get the configuration processed, otherwise it will return the configuration section as it.
