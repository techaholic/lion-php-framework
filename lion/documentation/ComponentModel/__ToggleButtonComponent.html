Class: __ToggleButtonComponent


Class Overview

Class Overview | Inherited Properties, Constants, and Methods | Properties Summary | Properties Detail | Method Summary | Methods Detail
A toggle button represents a button like a checkbox, with 2 possible status: pressed and not pressed.



A toggle button tag is "togglebutton"

i.e. 

  1.    <comp:togglebutton name="play_button" caption="Play" pressedCaption="Pause"/>
    2.

As a checkbox, the toggle button is a valueholder containing a boolean value. When the toggle button is pressed, value is true, otherwise value is false.

A toggle button has a caption, which is the text shown within the button.
A toggle button has a pressedCaption, which is the text shown within the button when it is pressed.


See the togglebutton component in action here: http://www.lionframework.org/components/togglebutton.html
