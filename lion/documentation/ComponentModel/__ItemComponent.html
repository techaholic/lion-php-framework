Class: __ItemComponent


Class Overview

Class Overview | Inherited Properties, Constants, and Methods | Properties Summary | Properties Detail | Method Summary | Methods Detail
Represents an item within a list of items.

Item tag is item

i.e. 

  1.    <comp:item text="More than 100 employees" value="5" selected="true"/>
    2.

An item has 3 properties:

  • text, the text representing the item, which will be the text shown within a list based component.
  • value, the value represented by the item.
  • selected, if the current item is selected within a list based component


An item component can be contained just in list based components.
At the same time, a list based component can only contain items, even if the list component inherit from a __IContainer class.


See:  __ItemListComponent, __ComboBoxComponent
