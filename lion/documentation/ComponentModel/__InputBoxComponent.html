Class: __InputBoxComponent


Class Overview

Class Overview | Descendants | Inherited Properties, Constants, and Methods | Properties Summary | Properties Detail | Method Summary | Methods Detail
Represents the typical text box as a one single line rectangle to write text inside it.

An inputbox is one of the most basic valueholders, where the value is the text written by the user

Inputbox tag is inputbox

i.e. 

  1.    First name: <comp:inputbox name="first_name"/>
    2.

To retrieve the input value, use the __InputComponent::getValue() method, while to set the text use the __InputComponent::setValue().
The event that is raised when the user change the focus (the most typical use case for inputboxes), is the blur event.


See the inputbox component in action here: http://www.lionframework.org/components/inputbox.html
