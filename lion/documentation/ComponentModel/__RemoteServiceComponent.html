Class: __RemoteServiceComponent

The RemoteService component is used to allow your javascript code to call to method defined within the server's event handler.


The remote service tag is remoteservice

i.e. 

  1.    <comp:remoteservice name = "client_foo"
    2. 

  2.                      method = "server_foo"/>
    3.

In this example, name is the client function name that will be generated in order to call to the server method.
method (or eventHandlerMethod) is the name of the method already defined within the event handler that will be executed as a result of calling the client function foo.

We may need to define a method foo within our event handler: i.e. 

  1.    class MyEventHandler extends __EventHandler {
    2. 

  2.  
    3. 

  3.    ...
    4. 

  4.  
    5. 

  5.        public function server_foo() {
    6. 

  6.            return 'hi, there';
    7. 

  7.        }
    8. 

  8.  
    9. 

  9.    ...
    10. 

  10.  
    11. 

  11.    }
    12.

In order to call to this method from the client side, just call to the client_foo function from your javascript code:

i.e. 

  1.    <script language='javascript'>
    2. 

  2.  
    3. 

  3.        //call to the remote service from the client side:
    4. 

  4.        client_foo();
    5. 

  5.  
    6. 

  6.    </script>
    7.

in the other hand, you are able to define a callback to be executed when the client receives the response from the server.
This callback will receive as parameter the value returned by your server-side event handler method.

i.e. 

  1.    <comp:remoteservice name = "client_foo"
    2. 

  2.                      method = "server_foo"
    3. 

  3.                    callback = "callback_foo"/>
    4. 

  4.  
    5. 

  5.    <script language='javascript'>
    6. 

  6.  
    7. 

  7.        function callback_foo(parameter) {
    8. 

  8.            alert(parameter);
    9. 

  9.        }
    10. 

  10.  
    11. 

  11.    </script>
    12.
This callback will show an alert popup with the value returned by the server event handler method. In our example, it will show the 'hello, there' string.


Our recomendation is to use the javascript component in the client side, so the example could be finally as following:

  1.    <comp:remoteservice name = "client_foo"
    2. 

  2.                      method = "server_foo"
    3. 

  3.                    callback = "callback_foo"/>
    4. 

  4.  
    5. 

  5.    <comp:javascript>
    6. 

  6.    {literal}
    7. 

  7.  
    8. 

  8.        function callback_foo(parameter) {
    9. 

  9.          alert(parameter);
    10. 

  10.        }
    11. 

  11.  
    12. 

  12.    {/literal}
    13. 

  13.    </comp:javascript>
    14. 

  14.  
    15. 

  15.    <input type = "button" caption = "Call to server_foo method"
    16. 

  16.                           onclick = "client_foo();"/>
    17.

See this example in action at http://www.lionframework.org/components/remoteservice.html
