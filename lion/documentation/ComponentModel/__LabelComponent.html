Class: __LabelComponent


Class Overview

Class Overview | Inherited Properties, Constants, and Methods | Properties Summary | Properties Detail | Method Summary | Methods Detail
The label component is used to show a text.

Labels are usefull to control text shown as the result of event executions (i.e. to change a text as the result of an ajax call)

Labels tag is label
i.e. 

  1.    <comp:label name="client_name" text="Antonio Parraga"/>
    2.

We can alter the label's text by setting the text property, i.e. 
  1.    $label = $this->getComponent('client_name');
    2. 

  2.    $label->setText('Carolina Kop');
    3.
