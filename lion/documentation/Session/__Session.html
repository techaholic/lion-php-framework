Class: __Session


Class Overview

Class Overview | Properties Summary | Properties Detail | Method Summary | Methods Detail
This class represents the session.

You can get a reference to the __Session instance by calling the __Context::getSession() method 

  1.  //Get the session:
    2. 

  2.  $session = __ApplicationContext::getInstance()->getSession();
    3. 

  3.  
    4. 

  4.  //Retrieve information from the session by reference:
    5. 

  5.  $my_data = $session->getData('my_data');
    6.
